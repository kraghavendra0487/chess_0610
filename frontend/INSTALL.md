# Installation Guide

## Required Packages

The React Chessboard examples require these packages:

- `react-chessboard` - The chessboard UI component
- `chess.js` - Chess game logic and move validation

## Installation Options

### Option 1: Windows (Recommended)
Double-click `install-packages.bat` to automatically install all required packages.

### Option 2: Manual Installation
Run these commands in the frontend directory:

```bash
npm install react-chessboard
npm install chess.js
```

### Option 3: One Command
```bash
npm install react-chessboard chess.js
```

## After Installation

Once packages are installed, run:
```bash
npm run dev
```

Then open your browser to see the chess examples!

## Troubleshooting

If you get import errors:
1. Make sure you're in the `frontend` directory
2. Run the installation commands above
3. Restart the development server with `npm run dev`
